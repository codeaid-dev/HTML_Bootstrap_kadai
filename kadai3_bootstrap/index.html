PHP機能概要

PHPの動作

PHPはWebサーバがクライアントへ返すレスポンスをプログラムが作成する役割のサーバサイドスクリプトである

テキストと数の動作

文字列は単一引用符で囲む方法、二重引用符で囲む方法、ヒアドキュメントで記述する方法がある

制御文字やプログラム予約文字を表示するにはエスケープが必要である

文字列は連結ができ、数値は演算ができる

変数を使って数値や文字列を格納し、演算や連結ができる

判定と繰り返し

判定も繰り返しもテスト式で真偽値（trueまたはfalse）を評価する

if() { ~ } else if() { ~ } else { ~ }、switch~case(), break, defaultを使って判定をする

for () { ~ }、while () { ~ }、do { ~ } while ()を使って繰り返しをする

配列の操作

配列は1つの変数に複数のデータをまとめて格納できる

配列は格納場所に名前を付けることができる

foreach()を使って配列のキーと要素を取り出しながら繰り返すことができる

関数とファイル

関数は処理（アルゴリズム）をまとめて定義して呼び出すことができるようにしたものである

関数は引数を持ち、値を返す

関数の集まりを別のファイルにし呼び出すことができる
